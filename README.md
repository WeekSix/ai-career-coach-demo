## Prerequisites

- Python 3.8+
- Google Gemini API key

## Setup

1. Clone the repository
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Create a `.env` file in the project root and add your Google API key:
   ```
   GOOGLE_API_KEY=your_google_api_key_here
   ```

## Running the Application

```bash
streamlit run app.py
```

## Usage

1. **Home**: Overview of available features
2. **Career Chat**: Get instant career advice
3. **Resume Review**: Upload and get feedback on your resume
4. **Career Assessment**: Take a short assessment for personalized career recommendations
